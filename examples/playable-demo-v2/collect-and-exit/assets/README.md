# Collect-and-exit demo assets

This fixture contains deterministic source-like local assets for the #340
Playable Demo Asset Refresh. `../asset-manifest.json` records the file hashes,
sprite atlas frames, tileset metadata, tilemap layers, and audio metadata used by
the AP1.9.1 validation smoke.

The files are intentionally tiny fixtures, not production art/audio and not
generated preview output. Generated runs, dashboard exports, screenshots,
temporary smoke outputs, and asset previews must remain outside this tracked
asset directory.
