# Source Patch Evidence Bundle v1 fixture

`source-patch-evidence-bundle.sample.json` is an inert, fixture-scoped audit
bundle for Source Mutation Preview v1. It links a source patch preview, file class
report, diff integrity report, sandbox dry-run report, test summary, review
decision, and forbidden-action notices. It does not apply patches, merge
branches, execute commands, or grant browser/dashboard/Studio write authority.
