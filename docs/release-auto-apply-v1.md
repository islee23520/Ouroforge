# Broadened Bounded Auto-Apply and Game-Scale Rollback v1

This contract extends the Milestone 22 trust gradient and existing rollback / kill-switch surfaces for Era H Milestone 44. It is not a new writer, runtime, release system, or browser/Studio authority.

The broadened tier may return `auto-applied` only when the embedded Milestone 22 request is low-risk, auto-apply eligible, high-confidence, all-gates-passing, in budget, autonomy-enabled, and rollback-backed; release-scale preflight, post-apply rerun, provenance-bundle, and compliance evidence refs are present; game-scale rollback is one command through the existing `ouroforge rollback` path; and the human release gate remains pending.

High-risk or source-affecting proposals fall back to manual review. An engaged kill switch halts autonomy. browser/Studio surfaces remain read-only; dashboard and cockpit inspection remains read-only. The contract executes no commands, writes no trusted files, releases nothing, and makes no production-ready, quality/fun, Godot replacement/parity, auto-merge, self-approval, reviewer-bypass, or autonomous-shipping claim. Generated runs/assets/content/release artifacts remain untracked unless fixture-scoped.

Governance anchors #1 and #23 remain open.
