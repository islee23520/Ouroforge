### Era A: Evidence-Native Foundation

These milestones preserve the original #1 sequencing and remain the foundation for all later work:

0. **Private Bootstrap** — project identity, initial docs, workspace layout.
1. **Harness Kernel** — Seed, Run, Ledger, Evidence, Journal, Verdict, CLI skeleton.
2. **Browser QA Worker Pool** — openchrome/CDP execution, screenshots, console, performance, traces, parallel workers.
3. **Minimal Browser Game Runtime** — 2D runtime, fixed timestep, scene loading, probe API.
4. **Scenario DSL and Evaluator** — executable scenarios, replay, assertions, verdict output.
5. **Evolve Loop** — failed evidence to mutation proposal, rerun, comparison.
6. **Safe Authoring Layer** — scene tree, inspector, live preview, evidence dashboard, review-gated edits.
7. **Engine Expansion** — renderer, asset pipeline, audio, animation, physics, native/export/plugin design gates as later scoped work.
